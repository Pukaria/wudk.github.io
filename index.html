    🏆 旗舰项目

    🔄 双交易所数据自动化稽核系统

    上交所数据自动化稽核系统 与 深交所数据自动化稽核系统 共同构成了完整的市场数据质量保障体系。

    • 数据源覆盖：8 个数据源（指数行情、融资融券、股票/基金/债券成交概况等）
    • 技术亮点：Selenium 动态渲染 + HTTP JSON 接口 + JSONP 解析 + CTE 窗口函数查询
    • 自动化通知：钉钉机器人实时推送执行结果与异常告警
    • 效率提升：每日稽核时间从 2-3 小时缩短至 5 分钟以内

    📌 查看上交所系统 · 查看深交所系统

    ⏰ 农林牧渔三周报时效监控体系

    覆盖畜产品周报、生猪及猪肉价格周报、猪鸡蛋料比价周报三大核心报告，实现数据时效的实时监控。

    • 智能日期提取：从正文中自动提取“采集日”，支持范围日期（如“6月2—8日”取结束日）
    • 双轨道精准通知：正常@采集人员确认，异常@管理负责人跟进，避免信息轰炸
    • 独立数据源适配：分别适配农业农村部和国家价格监测中心两个平台
