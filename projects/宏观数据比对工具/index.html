宏观数据自动比对工具

📌 项目背景

在日常数据处理工作中，经常需要对比不同日期生成的宏观数据报表，手动比对效率低且容易出错。本工具实现了两个日期（0604 vs 0603）（只需要修改脚本日期）数据表的自动化全量比对，快速识别数据差异，大幅提升工作效率。

🛠️ 技术栈

工具 用途
Python 3.x 核心开发语言
Pandas 数据读取、清洗、合并与比对
OpenPyXL Excel 文件读写支持

🧠 核心逻辑解读

程序的核心任务是基于唯一标识符（ZBDM），将两个日期的数据表进行全外连接合并，并逐行比对关键日期字段（JZRQ），最终输出一份包含比对结果标记的完整 Excel 报表。

具体流程如下：

  1. 文件检查：自动验证 0604 和 0603 两期数据文件是否存在于指定目录
  2. 数据加载：使用 Pandas 读取两个 Excel 文件
  3. 列名清洗：自动去除列名首尾空格，避免匹配失败
  4. 关键字段校验：确保 ZBDM（唯一标识）和 JZRQ（关键日期）两列存在
  5. 数据合并：以 ZBDM 为键，进行全外连接（how='outer'），保留两边的所有记录
  6. 差异标记：新增 JZRQ_same 字段，逐行比较两期数据的 JZRQ 是否一致
  7. 结果输出：生成结构清晰的 Excel 比对表，并自动统计一致/不一致的数量

📝 关键代码片段

import pandas as pd
import os

# 合并两个DataFrame，基于ZBDM进行全外连接
merged = pd.merge(df_0604, df_0603, on='ZBDM', how='outer', suffixes=('_0604', '_0603'))

# 添加 JZRQ_same 字段：比较两期数据的 JZRQ 是否相同
merged['JZRQ_same'] = None
both_exist = merged['JZRQ_0604'].notna() & merged['JZRQ_0603'].notna()
merged.loc[both_exist, 'JZRQ_same'] = merged.loc[both_exist, 'JZRQ_0604'] == merged.loc[both_exist, 'JZRQ_0603']

# 重新排列列顺序：ZBDM 置顶，JZRQ_same 紧随其后
cols = ['ZBDM', 'JZRQ_same'] + [c for c in merged.columns if c not in ['ZBDM', 'JZRQ_same']]
merged = merged[cols]

# 输出结果并统计
merged.to_excel(output_file, index=False)
true_count = (merged['JZRQ_same'] == True).sum()
false_count = (merged['JZRQ_same'] == False).sum()
print(f"JZRQ_same = True 的数量：{true_count}")
print(f"JZRQ_same = False 的数量：{false_count}")
