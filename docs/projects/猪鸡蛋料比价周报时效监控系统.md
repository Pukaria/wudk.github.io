# 猪鸡蛋料比价周报时效监控系统

## 📌 项目背景

国家价格监测中心（`jgjcndrc.org.cn`）每周发布《猪鸡蛋料比价》周报，反映**生猪、鸡蛋、饲料三大品类的比价关系**，是判断养殖盈亏的重要参考指标。

与前两个监控（畜产品、生猪）不同，该报告**不直接在正文中写明采集日**，而是通过发布日期推算截止日期（发布时间 - 2 天）。且报告发布平台为**国家价格监测中心官网**，数据源不同于农业农村部官网，需要独立的抓取逻辑。

本系统作为 **“农林牧渔三周报监控体系”** 的第三支柱，实现了：
1. 定时抓取国家价格监测中心列表页，获取最新文章标题和发布日期
2. 根据业务规则**计算截止日期**（发布日期 - 2 天）
3. 与数据库中该业务线（LRMBMX）的最新 `JZRQ` 进行比对
4. 根据比对结果，通过钉钉**精准通知**（正常不@任何人，异常@管理负责人）

## 📥 核心功能详解

### 1. 发布日期提取（XPath 精准定位）

```python
def fetch_latest_article_info():
    params = {"clmId": CLM_ID, "sclmId": SCLM_ID}
    resp = requests.get(LIST_URL, params=params)
    tree = html.fromstring(resp.text)
    
    # 标题：第一个包含 /detail? 的链接
    title_elem = tree.xpath('//a[contains(@href, "/detail?")]/text()')
    title = title_elem[0].strip() if title_elem else ""
    
    # 发布日期：根据网站结构的 XPath 定位
    pub_elem = tree.xpath('/html/body/div[2]/div/div/main/div[2]/div/div[2]/div/div[1]/div[1]/span/span/text()')
    if not pub_elem:
        # 备选：查找所有包含斜杠的 span
        pub_elem = tree.xpath('//span[contains(text(), "/")]/text()')
    
    publish_date = pub_elem[0].strip().replace('/', '-')
    return title, publish_date
```

### 2. 截止日期计算规则

报告正文中**不包含采集日信息**，截止日期需通过发布日期推算：

```python
def compute_deadline_from_publish_date(publish_date):
    pub_dt = datetime.strptime(publish_date, '%Y-%m-%d')
    deadline_dt = pub_dt - timedelta(days=2)  # 发布日期往前推 2 天
    return deadline_dt.strftime('%Y-%m-%d')
```

**业务逻辑说明**：该周报反映的是发布前 2 天（即周六至周一）的市场数据，因此截止日期 = 发布日期 - 2 天。

### 3. 差异化通知策略

与前两个监控不同，本系统在**正常时不@任何人**（避免信息过载），仅在异常时@管理负责人：

| 比对结果 | 钉钉消息标题 | @人员 | 理由 |
|:---|:---|:---|:---|
| 网页日期 = 数据库 JZRQ | ✅ `{LRMBMX} 数据已入库` | **不@任何人** | 正常情况，静默通知即可 |
| 网页日期 ≠ 数据库 JZRQ | ⚠️ `{LRMBMX} 数据疑似未入库` | `13624296576` | 需跟进处理 |

### 数据库查询（三表联查）

```python
def get_latest_jzrq_by_lrmbmx(self, lrmbmx):
    query = """
        SELECT TOP 1 CONVERT(varchar(10), A.JZRQ, 23) AS JZRQ
        FROM usrHYSJB A
        JOIN usrEDBZBGZB B ON A.ZBDM = B.ZBDM
        JOIN usrHYZBB C ON B.ZBDM = C.ZBDM AND A.JZRQ = C.JZRQ
        WHERE B.LRMBMX = ?
        ORDER BY A.JZRQ DESC
    """
    cursor.execute(query, (lrmbmx,))
    row = cursor.fetchone()
    return row[0] if row and row[0] else None
```
